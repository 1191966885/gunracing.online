Brain Match-3

Game Description

Brilliant Jewels is a Match 3 game where you can either play against the time or try to pass levels with limited amount of moves. If you are feeling confident, you can also challenge the hard mode on the time attack mode. This game doesn't lack any sparkle!

Instructions

Match three or more similar jewels to clear them from the board. Match four or more to receive special tiles. Combine two special tiles to create something even more special.
